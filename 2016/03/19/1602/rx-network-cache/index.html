使用 Rx 同步并缓存网络数据

发表于 2016-03-19   |   分类于 Android   |     |   阅读次数

RxJava是响应式编程, 在异步处理网络数据时, 使用广泛. 使用一些Rx的特性, 缓存网络数据, 并同步显示, 可以增强用户的网络加载体验.

更多: http://www.wangchenlong.org/

缓存模式: 读取数据库, 显示, 请求数据, 存储到数据库, 再更新页面.

使用Dagger2+Retrofit+Rx的标准组合, 我来讲解一下如何使用.
